# Données multimédia : Python pour le traitement d’images et de données audio

## Mini-Projet 

**À rendre avant le lundi 8 janvier 2024 sur Celene**

L’objet du mini-projet est le suivant : 
 1. rassemblez une dizaine de manifests IIIF,
 2. utilisez YOLOv8 et [ultralytics](https://github.com/ultralytics/ultralytics) pour détecter les objets contenus dans les images de ces manifests
 3. modifier les manifests pour y ajouter les objets détectés dans la section `metadata` et générez de nouveaux fichiers

Pour ce devoir vous devez reprendre des exercices ou des traitements vus en cours et les faire opérer dans une chaîne de traitements.  
Les documents attendus sont vos scripts ou notebook python, les fichiers d’entrée et de sortie et la documentation qui indique comment reproduire le résultat (doc technique, pas une dissertation).
